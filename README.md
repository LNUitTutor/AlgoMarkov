# Алгорифми Маркова

Простий інтерпретатор алгоритмів Маркова. Систему підстановок задають у текстовому файлі.

Файл підстановок має таку структуру:
* _n_ - кількість підстановок
* $підстановка_1$
* ...
* $підстановка_n$

Кожна підстановка складається з трьох слів: **стрілка** *ліва_частина* *права_частина*, де **стрілка** - це > (більше) для звичайної підстановки, або . (крапка) для кінцевої. Порожнє слово позначають \_/\\_
